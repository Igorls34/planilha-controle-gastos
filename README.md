# Planilha de Controle de Gastos - Landing Page

Landing page para captura de leads e download da planilha de controle de gastos.

## Como funciona

1. O usuário preenche nome, email e telefone no formulário
2. A API envia a planilha automaticamente por email e WhatsApp
3. O lead é registrado no servidor

## Tecnologias

- HTML/CSS/JS puro (sem frameworks)
- Design baseado no portfolio IgorDev
- API Node.js no backend

## Deploy

Basta fazer upload do `index.html` para qualquer servidor estático (Netlify, Vercel, GitHub Pages, etc).

O formulário se comunica com a API em `http://2.24.124.93:3002/api/capturar-lead`.
